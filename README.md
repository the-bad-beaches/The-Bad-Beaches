# The Bad Beaches

A 2D tower-defense game developed using [Godot Engine](https://godotengine.org/) v4.5 for the [GitHub Game Off 2025](https://itch.io/jam/game-off-2025).

## Overview

The base concept is that there are unknown underwater creatures that are attacking the beaches and you need to defend them with the help of various land creatures. The game will feature multiple levels, each with unique environments related to oceans and waves and enemy types.

## Gameplay

The exact gameplay mechanics have not been finalized yet, but is inspired by PvZ and some other tower-defense games.
